サウナで浮かんだアイデアを「忘れないようにリピートする」苦痛を排除し、**「手だけ動かしてAIに完結させる」**ための仕組みです。

---

## 1. ワークフロー全体図

| フェーズ       | 場所 / ツール  | アクション（手だけ動かす）          | 脳の状態           |
| :--------- | :-------- | :--------------------- | :------------- |
| **1. 発生**  | サウナ・外気浴   | 思考を流す（**リピートしない**）     | リラックス          |
| **2. 排出**  | 脱衣所 / スマホ | Geminiへ**音声入力**で羅列を投げる | **メモリ解放（空っぽ）** |
| **3. 構造化** | Gemini    | Keepへ「Code用指示書」を作成     | 自動処理           |
| **4. 実行**  | PC / Code | 指示書を**コピペ**して実行        | Vibe Coding    |

---

## 2. 各ステップの詳細

### ステップ1：排出（サウナ直後）
スマホを手に取れる状態になった瞬間に、記憶が消える前に「音声」でGeminiにすべて投げます。
* **トリガー：** 脱衣所のロッカーを開けた、または帰りのベンチに座った時。
* **アクション：** 「あとCodeにやらせたいこと。〇〇の機能を修正して、ついでにテストも書いておいて…」と、**文脈を気にせず口に出す。**
* **メリット：** タイピングの手間を省き、リピートし続ける苦痛（高コスト作業）を即座に解消します。

### ステップ2：整理（AIによる構造化）
Geminiが、断片的な音声入力をCodeが理解しやすい「依頼書形式」に変換します。
* **処理内容：**
    * 達成したいゴール（Goal）
    * 具体的なステップ（Task）
    * 開発環境や制約事項（Context）
* **保存場所：** Google Keep に保存。

### ステップ3：実行（PC前）
PCを開いた際、「何をするんだっけ？」と思い出すコストをゼロにします。
* **アクション：** Keepを開き、内容をコピーして Code にペーストするだけ。
* **プロンプト例：** `以下の指示書に基づき、このプロジェクトのコードを修正して。 [指示書の内容を貼り付け]`

## 💡 運用のポイント
* **リピート禁止:** サウナの中では「忘れてもいい」と自分に許可を出す。
* **テンプレ利用:** 管理作業を「手だけ動かす」ために、Geminiに「Keepに登録しておいて」と明示的に頼む。
