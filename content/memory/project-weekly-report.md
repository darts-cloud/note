# 週報自動化プロジェクト

**ステータス:** 着手待ち

## 概要

毎週WBSから手動で情報を拾って週報を書いているが、自動化したい。

## 仕様

- **方式:** 社内PCにPythonスクリプトを配置して実行
- **入力:** ExcelのWBSファイル（PV・EV・ACの列がある）
- **処理:** SV（EV-PV）・CV（EV-AC）を計算し分析
- **出力:** 以下4項目の週報草稿テキスト
  1. 遅延しているタスクとその理由（SV < 0）
  2. コスト超過の項目と対策（CV < 0）
  3. 全体の進捗率と予測完了日（EAC等）
  4. 今週の実績 vs 計画

## 注意事項

着手時に「社内PCでPythonが使えるか」を確認してからスクリプトを作成する。
